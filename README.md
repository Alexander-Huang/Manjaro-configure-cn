# Manjaro-configure-cn
一个帮助大陆用户配置Manjaro Linux18的脚本，让你体会到什么叫一把梭（手动滑稽）

用法：  
1.wget https://github.com/Alexander-Huang/Manjaro-configure-cn/archive/master.zip  
2.unzip master.zip  
3.sudo sh ~/Manjaro-configure-cn-master/autoconfig.sh
